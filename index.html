-How I work?-

Interwiew

I plan the project with the client and we create a "framework of the project" together. We discuss the details. How the site should look like, what functionalities the client needs.

Planning

Based on the "framework of the project" I make a plan how the site will be created. How long it will take to build the site. I present my plan to the client and make a sketch.

Coding

The next step is to code the site using HTML, CSS, JS and, if necessary, Bootstrap. I send the client information and screenshots about each completed milestone. When the site is ready, I present it to the client, present how it works and we discuss the possible changes.

Ready Website

The final step is uploading the site to the web, reviewing and testing it. In this step, I make final changes and discuss them with the client. If everything is in line with the client's expectations, we close the deal. In the first month, I also provide all small changes for free.
